•WINDYHOP TYPOGRAPHY

Font

Windyhop's official font is Parisian (Macintosh), and EllaFont (? ask Mary!) for PC. Until an appropriate secondary typeface is found, there is to be no substitution of any kind in direct juxtaposition with the Windyhop Logotype. In print, no typeface other than Parisian should appear in juxtaposition with the Windyhop logotype.

Due to Parisian's dramatic thick-thin nature, it is often necessary to stroke its points to a width varying on the size and nature of the medium. I am still developing a formula for this measurement of stroke witdth.

Usage of display type

Display type, or that which is created to call attention to itself as opposed to its contained information--thus, that which is generated in Illustrator or Photoshop and is a .jpg or .gif, is to only be used at the heading of default and subpages. It must be a minimum of 36 pt as the page header. No other usage or placement is acceptable. When display type is used, it is to be in the main branch color--the darker of the two colors.

Usage of text type

The appropriate format for page text layout is as follows:

CARDINAL RULES OF TEXT TYPE:

•Headlines and subheaders should not be more than one line of text long if it is possible to avoid.

•NO TAB INDENTS are to be used at the beginning of paragraphs under ANY circumstances.

•ONE hard return between paragraphs of text type.

•One hard return before a table or graphic and two directly following under all circumstances.

•TWO hard returns if preceding a headline, but only one if preceding a subheader.

•One SOFT return between Subheading and Body.

•Text type is to always appear in SEPIA, NOT BLACK!

•WINDYHOP TABLE DIMENSIONS

Table dimensions are flexible to a certain extent to accomodate odd data, but effort should always be made to create an obvious but at the same time downplayed presence of a border. The above border is set at 1. It should NEVER be set at more than 2. However, allow for ample cell padding, as the above table is set at 6 to ease readability, and cell spacing at 1.

In tables that are not co py-heavy, such as the above, it is appropriate to use a font size of 3. All headings within the table should always be bolded, and if applicable, X and Y may be differentiated by using the main and accent colors.

As a general rule, there should be NO clipart on any Windyhop page, to enforce and uphold a high visual aesthetic standard. The gallery may contain color photos but on all other main Windyhop pages, photos should be sepia-tinted with a Photoshopped white edging and vintage corner holders.
